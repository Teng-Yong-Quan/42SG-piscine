Write a program that takes one string as an argument.

For each alphabetic character (a-z or A-Z) that appears in the string, print the number of occurrences followed by the character.

Counting is case-insensitive (A and a count as the same letter).
The output order is the order in which each letter first appears in the string.
Ignore non-alphabetic characters.
Separate each result with ", " (comma and space).
End with a newline.
If the number of arguments is not 2, print only a newline.

Final Exam: Level 14
